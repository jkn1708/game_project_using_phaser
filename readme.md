# 🎮 Hobby Game Development Roadmap

프론트엔드 개발자로서(JavaScript / TypeScript / React) 취미로 게임
개발을 시작하기 위한 개인 로드맵이다.\
목표는 **작은 게임을 직접 완성하면서 게임 개발의 핵심 구조를 이해하는
것**이다.

------

# 1. 목표

## 단기 목표

- 브라우저 기반 2D 게임 1\~2개 완성
- 게임 루프 / 물리 / 충돌 / 상태관리 이해

## 중기 목표

- 게임 엔진 기반 개발 경험 (Unity or Godot)
- 실제 게임 구조(Scene, Entity, Physics) 이해

## 장기 목표

- 완성도 있는 개인 게임 1개 제작
- 모바일 또는 웹 배포

------

# 2. 개발 전략

프론트엔드 개발자의 강점을 활용하여 **웹 기반 게임 → 게임 엔진 → 3D
확장** 순서로 학습한다.

Web Game Development → Game Engine Development → Advanced Game
Development (3D / Multiplayer)

------

# 3. Phase 1 --- Web Game Development

## 사용 기술

- JavaScript
- TypeScript
- HTML5 Canvas
- Phaser Game Framework

### Phaser

Phaser는 HTML5 기반의 대표적인 웹 게임 프레임워크이다.

참고 링크

- 공식 사이트\
  https://phaser.io/
- 공식 튜토리얼\
  https://phaser.io/learn
- Phaser 3 Documentation\
  https://newdocs.phaser.io/
- GitHub Repository\
  https://github.com/photonstorm/phaser

------

## Phaser 기본 구조

Game ├ Scene │ ├ preload() │ ├ create() │ └ update()

핵심 개념

- Game Loop
- Scene 관리
- Sprite
- Physics
- Collision Detection
- Input Handling
